hyphenation
===========

Hyphenation patterns for Indian languages. mirror of http://git.savannah.gnu.org/cgit/smc/hyphenation.git


Linux packages
==============

------

These hyphenation rules are packaged in Debian. Following are the package names:

* hyphen-bn
* hyphen-pa
* hyphen-gu
* hyphen-as
* hyphen-mr
* hyphen-ta
* hyphen-te
* hyphen-ru
* hyphen-kn
